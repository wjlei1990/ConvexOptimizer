# ConvexOptimizer
C++ implementation of numerical optimizer for convex function in parallel environment.

** primitive version. Updates very soon...

### Dependancy
 1. boost (with c++11 std)
 2. boost_mpi
 
### Build
 1. make a build directory
 ```
 mkdir build
 ```
 2. cmake and compile
 ```
 cmake .. && make
 ```
